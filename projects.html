ASTRA - Aerospace Simulator

The Aerospace Simulation Software is a Java-based application that simulates the dynamics of aerospace vehicles. It provides a graphical user interface (GUI) using JavaFX to visualize the movement and parameters of the vehicles in real-time.
Add multiple aerospace vehicles with custom parameters. Simulate the dynamics of the vehicles including velocity and altitude changes. Real-time graphical visualization of the vehicle's movements. Controls to start, stop, reset, speed up, and slow down the simulation.
Java JavaFX
Java Development Kit (JDK) 11 or higher JavaFX SDK

DRASTIC - Distater Relief AI

Drastic is a AI that helps with calculating the needs for Goods, transport, and personnel needed for rescue/disaster situations. It is designed to calculate if the disaster relief will be a success or failure.

Drastic handles calculations for 3 main catagories.
Goods : Food / Goods / Items / Medicine
Personnel : Volunteers / Doctors / Engineers / etc
Transportation : Type / Cost / Carry-Capacity / # of type of vehicle

Drastic allows for a user to input various data for calculations.
Name of disaster
# of people effected (This may include injured, loss of power, loss of home, etc)
location of disaster (using google API for narowing down specific towns/cities)
# of days known and or expected for disaster to effect population

MIME - Voice Assistant

Mime voice assistant is a replication of Siri/Alexa.

Mime allows for a user to give audio input and recieve audio output.
The user can ask various questions regarding weather, interent search results, etc.

This program was designed using Python and various Python libraries that help create a chat model.

With simple changes to its training model and base requests, this program can be adapted to various industries as a helpful chat tool, as well as be used in a team setting to help with workflow.
I aim to improve its voice model to be more friendly. Additionally, I would want to create a higher sense of relationship between the program and its user to provide more meaninful and personal answers.

ColorNest - Color Code Manager

This program allows a user to store color codes and create color code templates that they can save for later use.
It runs with a C++ Backend and a Html, CSS, Javascript Frontend. This program tested by abilities by introducing new libraries. It also allowed for learning more about blending multiple languages to create a program. XAML files are new to me so that was a new stepping stone in the progression of this program.
This Color Code Manager program can be adapted for use by multiple industries such as Graphic Design, Artists, Makeup Artists, Interior Designers, etc. I would want to continue to improve the UI/UX of the program to be easier and cleaner for the user. This personal project was a learning curve with the blend of languages and was a fun challenge!
